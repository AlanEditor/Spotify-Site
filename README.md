# Clone Spotify

## Sobre o projeto
Este site foi criado para colocar em prática os conhecimentos adquiridos no módulo de **Front-End** do curso **Desenvolvimento Web Completo.**

O objetivo do projeto foi criar um clone do Spotify totalmente responsivo utilizando algumas das técnologias front-end, como HTML, CSS e Bootstrap
